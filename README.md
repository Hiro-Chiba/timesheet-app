# タイムシートアプリについて

出退勤とシフトを手軽に管理できるシンプルな Next.js アプリです。

## 使い方

1. 依存関係をインストールします。
   ```bash
   npm install
   ```
2. 開発サーバーを起動します。
   ```bash
   npm run dev
   ```
3. ブラウザで [http://localhost:3000](http://localhost:3000) を開き、アプリを操作します。

主に `app/` 配下のページと `components/` 配下のコンポーネントを編集すると、画面が自動で更新されます。
